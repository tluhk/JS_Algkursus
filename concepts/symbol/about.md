# Symbol

JavaScripti `Symbol` andmetüüp tähistab unikaalset identifikaatorit. Sümboleid kasutatakse sageli privaatseks mõeldud objektide atribuutide loomiseks või massiivi objektide jaoks ainulaadsete võtmete loomiseks.

```javascript
let id = Symbol();
```
[Allikas](https://developer.mozilla.org/en-US/docs/Glossary/Symbol)
